# CSharp_Prob_Sol
1. Data Structures
  1.1Create a list of integers and add 5 elements to it.
  1.2Create a dictionary with string keys and integer values, and add 5 key-value pairs to it.
  1.3Create a queue of integers and add 5 elements to it.
  1.4Create a stack of integers and add 5 elements to it.
2. Operations on Data Structures
  2.1Calculate and print the sum of all elements in the list.
  2.2Calculate and print the sum of all values in the dictionary.
  2.3Remove 2 elements from the queue and print the remaining elements.
  2.4Remove 2 elements from the stack and print the remaining elements.
3. Additional Tasks
  3.1Print all elements of the list.
  3.2From a three-digit number (e.g., 124), print the largest digit.
  3.3From a three-digit number (e.g., 124), print the smallest digit.
  3.4From an array, find all even values and print their sum.
  3.5Calculate and print the average of all elements in the array.
